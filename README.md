# EasyStartupAccount

Небольшой Go-сервис для управления аккаунтами. Написан в 2024, как pet-проект, код открыт в 2026 (обновлен только Dockerfile и README! Остальной код сохранен так, как он выглядел в 2024! Никакого рефакторинга с тех пор не производилось!).

### Технологии
- Go
- Rest API
- Gin
- JWT
- SQL (MySQL)
- Swagger

### Быстрый старт

Требования: `Go >= 1.22`, Docker (опционально).

Сборка локально:

```bash
go build -o esaccount ./cmd/release
./esaccount
```


### Переменные окружения

- `ADMIN_TOKEN` - токен администратора (используется в `pkg/service/auth.go`).
- `DB_PASSWORD` - пароль для ДБ

Все остальные параметры хранятся в configs/config.yml

### Конфигурация
- Для запуска нужно прописать конфигурационный файл configs/config.yml. Пример ключевых полей:

```yaml
port: "8011"
db:
	host: "..."
	port: "3306"
	username: "..."
	dbname: "..."
```
